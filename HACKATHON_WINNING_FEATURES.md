## 🚀 **Quick Start Guide**

### **1. Backend Setup:**
```bash
cd backend
npm install
npm run dev  # Starts on port 5000
```

### **2. Frontend Setup:**
```bash
cd frontend
npm install
npm run dev  # Starts on port 3001
```

### **3. Environment Variables:**
```env
GOOGLE_AI_API_KEY=your_gemini_api_key
OPENAI_API_KEY=your_openai_api_key
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_jwt_secret
```

### **4. Test the New Features:**
1. **AI Tutor:** Visit any course page and click "AI Tutor" button
2. **Study Rooms:** Go to collaboration section and create a room
3. **Analytics:** Admin dashboard → Advanced Analytics
4. **Accessibility:** Click the accessibility button (bottom right)
5. **Content Generation:** Instructor tools → Smart Content
